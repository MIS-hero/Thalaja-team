Rounded surface container for grouping content; the `pop` variant carries the brand's hard-offset shadow.

```jsx
<Card variant="pop" padding="md" radius="lg">
  <h3>Weekend run</h3>
  <p>12 items · shared with 4</p>
</Card>
```

Variants: `plain` (white + soft hairline, default), `pop` (navy outline + hard shadow — use for hero/featured cards), `sunken` (deeper cream, for nested groups), `brand` (red fill, cream text). Add `tone` for a tinted pastel fill. Keep `pop` for one or two emphasis cards per screen, not every row.
