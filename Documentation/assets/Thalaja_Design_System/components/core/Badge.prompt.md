Small pill for statuses, counts, and category labels.

```jsx
<Badge tone="mint" variant="soft" dot>In stock</Badge>
<Badge tone="red" variant="solid">3</Badge>
```

Tones: `neutral | red | mint | indigo | lilac | navy`. Variants: `soft` (tinted, default), `solid`, `outline`. Use `dot` for live-status. Sizes `sm | md`. Reach for `mint` = good/done, `red` = needs attention/out, `indigo` = informational.
