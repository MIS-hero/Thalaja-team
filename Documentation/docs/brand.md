# Brand Reference — Thalaja

## Color Tokens

| Name | Hex | Role |
|---|---|---|
| **Tomato** | `#FF4924` | **Primary** — buttons, CTAs, logo fill, key actions |
| Night | `#0D0050` | Text, dark backgrounds, logo stroke |
| Cream | `#FFF8ED` | App background |
| Mint | `#00EE8E` | Accent, success states, ث dot motif |
| Peri | `#4444D5` | Secondary, app icon background |
| Lavender | `#F3CDEA` | Soft accent |

**Tomato is the primary color. Night is text/dark.** These are the most frequently confused pair.

## Typography

- **Typeface:** Cairo Black (Arabic)
- **Direction:** RTL-first. Arabic is the primary language — not a translation layer added on top of an LTR layout.

## Logo System

- **Shape:** Organic 8-point rounded star
- **Fill:** Tomato `#FF4924`
- **Stroke:** Night `#0D0050`, thick
- **Dot motif:** Three Mint `#00EE8E` rounded squares — form the ث letter dots; also read visually as orange fruit dots (intentional dual meaning)
- **App icon:** Star on Peri `#4444D5` background

## Category Sticker System

Used for grocery category labels throughout the UI.

- Shape: horizontally stretched 8-point star, approximately 3:1 width-to-height ratio
- Fill: unique palette color per category (not fixed to one token)
- Left zone: illustration of the category
- Right zone: Arabic category name, RTL-aligned
- ث dot motif: integrated only where it emerges naturally from the illustration concept — never forced

## Aesthetic Reference

Retro fruit sticker aesthetic: octagonal badges, scalloped shapes, circular perimeter text. UI components reflect this visual language. Do not default to flat-minimal system components without considering whether the aesthetic applies.
